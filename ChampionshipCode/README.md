# 2019DeepSpace
This repo holds the Howdy Bot's code for the robot created to solve the 2019 FRC Challenge: Deep Space.

# Requirements
- LabVIEW FRC 2019 (using LabVIEW 2018)
- FRC Update Suite 2019
- CTRE Phoenix Framework
